# Crypto Law Source Map

Primary-source-first retrieval guide for Anthony's US crypto and stablecoin research.

**Scope:** US federal law plus selected state overlays for startup operations  
**Selected states:** Florida, Michigan, Delaware, New York comparator only  
**Last verified:** July 14, 2026  
**Use:** Load this file first, then load the narrower deep-dive file that matches the user's issue.

## Retrieval Order

1. Confirm whether the question is about issuance, trading, custody, payments, reporting, enforcement, or entity structuring.
2. Use the table below to pull the highest-authority source first.
3. Only after primary sources, consult secondary commentary if the user needs synthesis, market practice, or implementation sequencing.

## Authority Key

- **Primary**: statute, regulation, formal Commission/agency interpretation, court opinion, official state source
- **Secondary-Gov**: CRS, White House fact sheet, agency press release describing an official action, **staff** statements
- **Secondary**: law firm, industry, academic, trade publication

**Securities hierarchy (do not invert):** statute (including GENIUS § 17 once effective) → Commission interpretive release → staff statements / speeches → secondary commentary.
